# NextGen Store

NextGen Store is a modern, full-featured e-commerce front-end built with Next.js and React. It provides a seamless shopping experience, allowing users to browse products, add items to a cart, and manage their profile. The project leverages the power of Next.js for a fast, server-rendered application.

## Key Features

-   **Product Catalog:** Browse a list of available products.
-   **Shopping Cart:** Add/remove items and view the cart total.
-   **User Profile:** A dedicated page for user information.
-   **Responsive Design:** The UI is optimized for both desktop and mobile devices using CSS modules.
-   **Modern Tech Stack:** Built with the latest features of Next.js 14 (App Router) and React.
-   **Optimized Loading:** Includes loading states for a better user experience while data is being fetched.
-   **Custom 404 Page:** A user-friendly "Not Found" page for invalid routes.

## Technologies Used

-   [Next.js](https://nextjs.org/)
-   [React](https://reactjs.org/)
-   CSS Modules
-   ESLint for code quality

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

-   Node.js and npm installed on your machine.

### Installation & Running

1.  Clone the repo or download the source code.
2.  Install NPM packages:
    ```sh
    npm install
    ```
3.  Run the development server:
    ```sh
    npm run dev
    ```
4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
